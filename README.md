Glitch-flower Grubbs

Welcome to Glitch's and Sunflower Grubbs, the ultimate recipe search site! Whether you're a seasoned chef or a kitchen newbie, our platform is designed to help you find and save the perfect recipes to satisfy your cravings.

Features

1. Search Recipes

Easily search for recipes by ingredient, cuisine, or dish type. Our smart search ensures you find exactly what you need, tailored to your preferences.

2. Recipe Details

Each recipe includes:

A list of ingredients.

Step-by-step instructions.

Estimated prep and cooking time.

Nutritional information.


3. User-Friendly Categories

Explore recipes sorted by categories such as:

Breakfast

Lunch

Dinner

Desserts

Vegetarian/Vegan

Quick Meals


4. Save and Share

Bookmark your favorite recipes for easy access.

Share recipes with friends and family through social media or email.


5. Interactive Features

Comment on recipes to share tips and feedback.

Upload your own recipes to the community.


Tech Stack

Frontend: React vite 

Backend: Node.js (Express.js) or Django/Flask

Database: Edaman

Hosting: Vercel

Version Control: Git


Installation

1. Clone this repository:

git clone https://github.com/username/recipe-site.git


2. Navigate to the project directory:

cd recipe-site


3. Install dependencies:

npm install  # or pip install -r requirements.txt


4. Start the development server:

npm start  # or python manage.py runserver
